1. Interactive TUI (Live Preview Mode)
The TUI allows you to type Arabic text and see the KuKufi representation generated live. It supports colored output, framing, and letter spacing dynamically.
To launch the interactive TUI, run:
cargo run -- interactive
(Press Ctrl+C or Esc to exit the interface)
You can also pass modifiers to the TUI mode so the live preview is colored, framed, or spaced out:
cargo run -- --color --frame --spacing 2 interactive
2. Standard Text Conversion
If you just want to pass Arabic text directly from the terminal and get the Kufi output printed to standard output:
Basic render:
cargo run -- "بسم الله"
With color and Unicode borders:
cargo run -- --color --frame "بسم الله"
With horizontal spacing between letters (e.g., 2 spaces):
cargo run -- --spacing 2 "بسم الله"
3. File Output Generation
You can output the ASCII art to a text file. If you use the .ans extension, it will automatically embed ANSI color escape sequences so you can cat the file later and see the colors.
# Saves plain ASCII art
cargo run -- --output basmala.txt "بسم الله الرحمن الرحيم"
# Saves colored ASCII art (notice the .ans extension)
cargo run -- --output basmala.ans --color "بسم الله الرحمن الرحيم"
(You can verify the colored output by running cat basmala.ans)
4. Reading from Standard Input (Piping)
If you want to pipe Arabic text generated by another script or tool directly into KuKufi:
echo "بسم" | cargo run -- --color
5. Developer & Debugging Tools
KuKufi comes with several built-in utilities to inspect your TOML dictionary (assets/glyphs.toml).
List all mapped glyphs:
cargo run -- list
Check the dictionary for structural issues or height mismatches:
cargo run -- validate
Show the isolated, initial, medial, and final forms of a specific letter:
cargo run -- show ba
cargo run -- show lam_alif
